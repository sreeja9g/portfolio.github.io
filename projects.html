    Title       :  Autism Spectrum Disorder
    
    Description :

Autism Spectrum Disorder (ASD) is a neurodevelopment disorder condition.We propose a new data set related to autism spectrum disorder detection of children and adults. In this data set we record some behavioural features plus some individual characteristics that have proved to be effective in detecting the ASD cases.For this we are using machine learning with python programming includes HTML, CSS, SQL data set for front end and back end.
